## Lecture Information  

Date: 4 April 2026  
ဆရာမ ခိုင်ဆုဝေ (Assistant Professor, Akita University Japan) က သူ့မဟာဘွဲ့အတွက် လုပ်ခဲ့တဲ့ သုတေသနရဲ့ တစ်ပိုင်းဖြစ်တဲ့ string similarity နဲ့ ပတ်သက်ပြီး လက်ချာပေးခဲ့တယ်။   

သူ့ လက်ချာ အပြီးမှာ နောက်ဆက်တွဲ အနေနဲ့ Myanmar sign language, Myanmar fingerspelling ပြီးတော့ SignWriting အပိုင်းတွေကို ဆရာနဲ့ ဆရာ့ကျောင်းသားတွေ လုပ်ခဲ့တဲ့ သုတေသနတချို့ကိုတွဲပြီး ဆရာ မိတ်ဆက်ပေးခဲ့တယ်။
