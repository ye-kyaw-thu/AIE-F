ဆရာမ ခိုင်ဆုဝေ (Assistant Professor, Akita University Japan) က သူ့မဟာဘွဲ့အတွက် လုပ်ခဲ့တဲ့ သုတေသနရဲ့ တစ်ပိုင်းဖြစ်တဲ့ string similarity နဲ့ ပတ်သက်ပြီး လက်ချာပေးခဲ့တယ်။
