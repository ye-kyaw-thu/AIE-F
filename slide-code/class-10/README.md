# For Your Information

- Class-10 မှာ RDR နဲ့ ပတ်သက်ပြီး မိတ်ဆက်ပေးခဲ့တယ်။ Slide က [https://github.com/ye-kyaw-thu/SCRDR_tutorial/blob/main/slide/RDR_Intro.pdf](https://github.com/ye-kyaw-thu/SCRDR_tutorial/blob/main/slide/RDR_Intro.pdf) ကနေ download လုပ်ယူလို့ ရတယ်။  
- ပြီးတော့ SCRDR နဲ့ ပတ်သက်ပြီး rule တွေကို domain expert အနေနဲ့ ဘယ်လို သတ်မှတ်လို့ ရတယ်၊ တကယ်လို့ ဒေတာက တော်တော်များများရှိပြီးသား ဆိုရင်တော့ automatic rule induction လုပ်လို့ ရကြောင်း၊ ပြီးတော့ `scrdr_tagger.py` နဲ့ `scrdr_tokenizer.py` ကိုသုံးပြီး POS tagging, word tokenization တွေကို လက်တွေ့ experiment ကို သုံးနာရီခွဲခန့် လုပ်ပြခဲ့တယ်။  
- Tutorial နဲ့ ပတ်သက်တဲ့ code, notebook, data, log တွေက ဒီ [https://github.com/ye-kyaw-thu/SCRDR_tutoria](https://github.com/ye-kyaw-thu/SCRDR_tutorial) repository မှာ အကုန် တင်ပေးထားပါတယ်။
